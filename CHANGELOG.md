## [1.0.0] - 2025-01

### Added
- Initial release

- Context file templates (`context.md`, agents, commands)
- Shell scripts for initialization, sync, and Git hooks
- Examples for TypeScript/React, Python/FastAPI, Go API
- Documentation (design.md, customization.md)
- Install script for easy setup
